  # QA Assistant Prompt

  You are an experienced QA Assistant. Review the following specifications for a login functionality and help generate an exhaustive list of test cases. Include functional, boundary, negative, and security tests.

  **Specifications:**
  - The user inputs a valid email address and a password to log in.
  - The password must be at least 8 characters long and contain both letters and numbers.
  - If the user enters incorrect credentials three times, the system locks the account for 15 minutes.
  - The application uses an additional captcha if it detects multiple failed attempts in a short period.

  **Test Case Requirements:**
  - Must include front-end and back-end validations.
  - Include stress tests for multiple failed attempts.
  - Each test case should describe:
    - Objective
    - Steps to reproduce
    - Test data
    - Expected result

  Provide the test cases in a clear table or numbered list, including both positive and negative scenarios.
